# prometheus-grafana-presentation
Repositório com exemplos de custom instrumentation no prometheus integrado ao Grafana. 

# run in local mode
## build aplication 
```shell
make install
```

## up all containers
```shell
make up
```

## up grafana
```shell
make up-grafana
```

## up api
```shell
make up-api
```

## down all containers
```shell
make down
```
